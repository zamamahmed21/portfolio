Sleep Scheduler

Problem Statement :

The client needed an easy-to-use desktop application to automatically schedule and put their PC into sleep mode at specific dates and times. The application needed to offer a user-friendly interface for scheduling tasks and operate reliably in the background without requiring constant user interaction.

Technologies Used

  • Python
  • PySide    (for creating the desktop interface)
  • Task Scheduler    (to manage sleep timing on Windows OS)
  • DateTime Module    (to manage dates and time schedules)

Key Features:

  • Allows users to set specific dates and times for the PC to enter sleep mode automatically.
  • Provides an intuitive and simple-to-use interface for scheduling sleep events.
  • Supports both one-time and recurring schedules for daily/weekly system sleep tasks.
  • Runs in the background to avoid disrupting the user's workflow.
  • Offers a system tray icon to monitor and modify schedules without reopening the application.

Challenges:

The biggest challenge was ensuring that the application could handle background tasks effectively while keeping system resources low. Additionally, providing precise scheduling and ensuring compatibility with different Windows versions was key.

Solution:

I used Python and PySide to create a lightweight desktop application with a clean interface. The Windows Task Scheduler was integrated using Python's `subprocess` module to manage background tasks. To allow the app to operate without user interaction, I designed it to run as a background process with a system tray icon for easy access. The DateTime module was used for accurate time-based task scheduling.

Outcome:

The application successfully automates PC sleep schedules based on user-defined times and dates, providing a reliable solution for users who want to save power or maintain a regular work schedule. The client was satisfied with the ease of use and performance of the Sleep Scheduler app.
