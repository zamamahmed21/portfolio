PDF Invoice Generator

Problem Statement :

The project aims to develop a user-friendly desktop application that simplifies invoice creation, customization, and management with features like input validation, PDF generation, and invoice history tracking.

Technologies Used

  • Python
  • PySide
  • fpdf   (Python library for generating and manipulating PDF documents programmatically.)
  • pdf2image   (Python library that converts PDF documents into images and display of PDF content as image files.)
  • XML   (A flexible markup language used for structuring, storing, and transporting data in a human-readable format.)

Key Features:

  • Users can input comprehensive details for each invoice and add items as needed.
  • Option to choose between two different PDF invoice templates for customization.
  • Users can view an updated image preview of the invoice before saving it.
  • Users can save up to five previous invoices for easy access and reuse at any time.

Challenges:

Displaying the preview of the pdf into the app.

Solution:

Since displaying a PDF directly in a PySide application is not feasible, I implemented a solution to convert the PDF into an image first. This conversion allows the PDF pages to be rendered as images, which can then be displayed in the application. Specifically, I used the 'pdf2image' library to convert the PDF into an image format. Once converted, the image is set to a QLabel within the app, enabling users to visually preview the invoice before saving it. This approach enhances the user experience by providing a clear and accurate representation of the invoice.

Outcome:

Allows users to effortlessly enter details, view a preview before generating the invoice, and conveniently access previously saved history for easy retrieval.
